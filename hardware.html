    Central Processing Unit (CPU)

    Another thing that is important to know is what the CPU is. It is the abbreviation for central processing unit and it basically is the brains of the computer. The CPU's history starts wtih the 8088 or eight eighty eight. The IBM (Internation Buiness Machines) decided to chose Intel's 8088 processor as the brain for the first computer. This made intel the main company in the CPU market and they continued to develop their CPUs even to present day.









    Random Access Memory (RAM)

    Another key term would be RAM. RAM is the acronym for Random Access Memory and it is a type of computer memory that can be accessed randomly and it is the most common type of memory found in computers. Before RAM computers just relays, mechanical counters or delay lines for memory. The first form of RAM was the Williams Tube in 1947. The way it stored data was it used electronically charged spots on the face of a cathode ray tube to store the data and since the beam of the cathode ray tube could write and read spots in any order the memory was randomly accessed. Another type of RAM that was developed in 1947 was the Magnetic-Core Memory which became a widespread form of random-access memory and it stores memory relying on an array of magnetic rings. Magnetic-Core Memory kept developing until the mid 1970's.







    Hard Drive

    The hard drive of your computer is a high capacity, self-contained storage device on your computer. It basically retrieves and stores information you saved. The hard drive was created in 1950 to make the computer better by having more storage space. The commercial usage of hard drives began in 1956.












    Motherboard

    The motherboard by itself is useless but it is needed for a computer to work. It's main job is to hold the computer's microprocessor chip and let everything else connect to it. The shape and layout of a motherboard is called the form factor. The form factor affects where individual components go and the shape of the computer's case. These components included Peripheral Component Interconnect (PCI)-connections for video, sound and video capture cards, as well as network cards. Accelerated Graphics Port (AGP) - port for video cards. Integrated Drive Electronics (IDE) - interfaces for the hard drives. Universal Serial Bus or FireWire - external peripherals and Memory slots
